🗺️ Bhu Naksha kya hota hai?

Simple Explanation: "Bhu Naksha = Zameen ka Digital Map"

Jaise Google Maps par roads aur buildings dikhti hain, waise hi Bhu Naksha par aapki zameen ka exact shape aur location dikhta hai.
Har plot ka ek Khesra number hota hai aur Bhu Naksha mein woh number se plot dhundh sakte hain.

What you can see in Bhu Naksha (नक्शे में क्या दिखता है):

✅ Apne plot ki exact boundaries (चौहद्दी / सीमा)
✅ Plot ka shape (square / rectangular / custom)
✅ Adjacent plots kaun se hain (आस-पास के प्लॉट)
✅ Roads aur water bodies nearby (नदी, नाला, रास्ता)
✅ Plot ka Khesra number (खेसरा संख्या)
✅ Approximate area / Raqba (रकबा)

Bhu Naksha Kab aur Kyun Zaroori Hai? (6 Major Use Cases)

🏗️

Construction (मकान निर्माण)

"Ghar banane se pehle plot boundaries check karein taaki kal ko koi encroaching ka arop na lagaye."

⚖️

Seema Vivad (सीमा विवाद)

"Padosi se seema vivad mein legal evidence ke roop mein Bhu Naksha kaam aata hai."

💰

Kharidna / Bechna (खरीद-बिक्री)

"Zameen kharidne se pehle exact location aur surrounding verify karein."

📄

LPC Application

"LPC application aur Revenue verification mein plot map helpful hota hai."

🌾

Agriculture (खेती-बाड़ी)

"Khet ki exact boundaries aur bandh (मेढ़) ki sahi position jaanne ke liye."

🏦

Bank Loan (बैंक ऋण)

"Property verification aur mortgage valuation ke liye bank map maang sakta hai."

MAIN OFFICIAL PORTAL

Best for:

Digital Cadastral Maps & Interactive Vector Search

Key Features:

  • ⚡ Khesra number se direct plot search
  • ⚡ Plot boundaries clearly visible & color coded
  • ⚡ Interactive zoom in / out vector maps
  • ⚡ Mobile friendly & fast loading

📋 Bhu Naksha Online Kaise Dekhen — Step by Step

eMapi official portal se 9 aasan steps mein naksha dekhen

eMapi Portal Kholen ↗
1

emapi.bihar.gov.in open karein

"Chrome browser use karein best experience ke liye."

2

District select karein

"Dropdown mein apna Jila chunein — Bihar ke sabhi 38 districts available hain."

3

Anchal (Block) select karein

"Apna Block / Anchal chunein jahan zameen sthit hai."

4

Halka select karein

"Apna Revenue Halka (हल्का संख्या) chunein."

5

Mauja (Village) select karein

"Apna gaon / mauja name list se chunein."

6

Map load hoga

"Aapke area ka map screen par aayega — sabhi plots ke Khesra numbers dikhenge."

7

Khesra number se search karein

"Search box mein apna Khesra number enter karein — woh specific plot map par highlight ho jayega."

8

Plot click karein

"Plot par click karne par uski details dikhegi: Khesra Number, Area/Raqba, Land Type (Awasiya/Krishi)."

9

Download / Print karein

"Print icon ya Download button se map ko PDF ya Image file mein save kar sakte hain."

💡 Pro Tips for Fast Bhu Naksha Search:

  • ✅ Khesra number pehle se ready rakhein
  • ✅ Chrome browser use karein
  • ✅ Fast internet connection hona chahiye
  • ✅ Mobile par desktop mode ON karein
  • ✅ Zoom in karein clearer plot boundary ke liye

🎨 Bhu Naksha mein Colors ka Matlab

Digital Bhu Naksha par alag-alag kism ki zameen ke liye alag rang hote hain. Color chart dekhen:

Color (रंग) Zameen ka Prakar (प्रकार) Description
Green Agricultural Land (कृषि भूमि) Kheti-bari waali zameen (Crop land)
Yellow Residential (आवासीय) Ghar / Awas banane ki zameen
Red Commercial (व्यावसायिक) Dukan, godown, commercial complex
Blue Water Body (जल स्रोत) Nadi, Nala, Pokhar, Talab, Canal
Grey Road / Path (रास्ता / सड़क) Sarak, gali, aal (आल / मेढ़)
Orange Government Land (सरकारी भूमि) Gair Mazarua Khas/Aam, Sarkari Zameen
Pink Forest Land (वन भूमि) Jangal / Forest Vibhag ki zameen

"Agar aapki zameen ka color galat dikh raha hai toh Parimarjan portal par correction apply karein."

Parimarjan Portal ↗

📥 Bhu Naksha Download Kaise Karein?

METHOD 1

From eMapi Portal

  1. Plot dhundhen (steps upar)
  2. Plot select karein
  3. "Print" ya "Download" click
  4. PDF ya Image save karein
METHOD 2

Screenshot

"Mobile ya Computer par screenshot le sakte hain — yeh informal use aur local dikhane ke liye kaafi hota hai."

METHOD 3

Certified Legal Copy

"Official / Legal use ke liye: Apne Anchal Office ya Survey Office se certified Bhu Naksha copy lein.
Fees: ₹50 - ₹200"

⚠️ IMPORTANT NOTE: Online downloaded map court ya registry mein directly legal proof ki tarah valid nahi hota. Court case ya official registry ke liye certified map copy Anchal / Survey office se prapt karein.
